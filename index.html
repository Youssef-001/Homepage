About me

Killua Zoldyck is a prominent character from the anime and manga series Hunter x Hunter by Yoshihiro Togashi. He is introduced as a young, talented assassin from the infamous Zoldyck family, known for their extraordinary skills in assassination. Despite his formidable background, Killua leaves his family in search of a different path, seeking freedom and adventure.

My Work

Battleship

A classic two-player grid-based strategy game where players place ships and take turns guessing the location of their opponent's ships.

Todo App

A task management app where users can create projects and add to-dos with priority levels. Built using JavaScript, HTML, and CSS, the app organizes tasks by project and priority, highlighting efficient task tracking and management. It demonstrates my ability to manage data and build modular, scalable code.

Weather Forecast

A weather application that allows users to search for any city and view the current weather conditions along with a 7-day forecast. Built using JavaScript, HTML, and CSS, it integrates with a weather API to provide accurate and real-time data. This project highlights my skills in API integration, asynchronous JavaScript, and responsive design.

Big Integer

A custom C++ library for handling large integer arithmetic beyond standard data type limits. The library supports addition, subtraction, multiplication of arbitrarily large integers.

Connect 4

A terminal-based implementation of the classic Connect Four game, developed in C++. This project features a two-player mode where players alternate turns to connect four discs in a row, either horizontally, vertically, or diagonally. The game includes basic input handling, win detection, and an intuitive text-based interface.
